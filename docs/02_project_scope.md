# Project Scope: Ecommerce Churn Prediction

## In Scope
* **Customer-level Churn Prediction:** Binary classification (Churn vs. No-Churn).
* **RFM Analysis:** Calculating Recency, Frequency, and Monetary value for every customer.
* **Behavioral Pattern Analysis:** Identifying trends in purchasing hours and quantities.
* **Dashboard Deployment:** A Streamlit interface for stakeholders to upload data and get predictions.

## Out of Scope
* Real-time "live" stream predictions (Batch processing only).
* Product recommendation engine (Collaborative filtering).
* Inventory and Supply Chain optimization.

## Timeline
* **Duration:** 8 Weeks (Data Acquisition to Deployment).

## Constraints
* **Tools:** Must use open-source libraries (Pandas, Scikit-learn, Docker).
* **Infrastructure:** No paid cloud APIs (Vertex AI/AWS SageMaker). 
* **Deployment:** Publicly accessible via Streamlit Cloud.
