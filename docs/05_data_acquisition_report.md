# Data Acquisition Report

## Source Details
* **Dataset:** Online Retail II
* **Provider:** UCI Machine Learning Repository
* **Format:** .xlsx (Excel)
* **Original Size:** ~45MB

## Retrieval Process
The data was programmatically downloaded using the `requests` library in `src/01_data_acquisition.py`. 

## Initial Findings
* **Total Records:** 525,461
* **Features:** 8 columns (Invoice, StockCode, Description, Quantity, InvoiceDate, Price, Customer ID, Country).
* **Time Span:** December 2009 to December 2010.
